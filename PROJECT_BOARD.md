# Project Board (Template)

Starting point for a GitHub Project or Trello board.

Columns:
- Backlog
- Milestone 1 – Proposal & Skeleton
- Milestone 2 – Alpha Prototype
- Milestone 3 – Beta & Evaluation
- Final – Demo & Report

- Finalize proposal text
- Implement HMAC chain
- Create synthetic log generator
- Implement corruption tests
- Build verification tool
- Run performance evaluation
- Prepare final report and demo
